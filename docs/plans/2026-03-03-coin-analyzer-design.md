# Coin Analyzer (counter-trade6.html) 설계

## 목적
거래량/변동률 상위 5개 코인을 선정하고, 각 코인별로 심층 기술적 분석 + 6전략 백테스트를 수행하여 코인별 최적 전략을 추천하는 독립 페이지.

## 코인 선정 로직

1. Bithumb KRW 마켓 전체 조회
2. 스테이블코인(USDT, USDC 등) 제외
3. 유효숫자 4자리 이상 필터 (1,234원 OK, 0.5432원 OK, 12원 X)
4. 정렬: `거래대금 × |24h변동률|` 상위 5개

## 심층 분석 지표

| 지표 | 파라미터 | 표시 |
|---|---|---|
| RSI | 14 | 값 + 과매수/과매도 배지 |
| Bollinger Bands | 20, 2 | 상단/중앙/하단 + 현재가 위치(%) |
| MACD | 12, 26, 9 | 히스토그램 + 골든/데드크로스 |
| EMA | 10, 20, 60, 100 | 4개 EMA 배열 상태 (정배열/역배열) |
| Stoch RSI | 14, 14, 3, 3 | K/D 값 + 크로스 |
| 거래량 추세 | 10봉 | 최근 거래량 vs 평균 비율 |
| 캔들 변동성 | 10봉 | 평균 레인지 % |

## 전략 매칭

- 각 코인에 6전략(RSI반등, BB반등, EMA크로스, MACD크로스, StochRSI, 복합) 백테스트 수행
- btSimulate 로직 재사용 (고가/저가 정밀, 손절 1.2배, 트레일링 목표도달 후)
- 총수익률 1위 전략을 추천 전략으로 표시

## UI 구성

### 탭1: 코인 선정
- "스캔" 버튼 → 필터링 → 상위 5개 카드 표시
- 카드: 심볼, 이름, 현재가, 24h변동률, 거래대금
- "분석 시작" → 탭2 이동

### 탭2: 심층 분석
- 상단: 5코인 서브탭
- 중단: 지표 대시보드 (배지/값) + Lightweight Charts 캔들차트 (BB/EMA 오버레이)
- 하단: 6전략 백테스트 결과 테이블 + 추천 전략 하이라이트

### 탭3: 설정
- Bithumb API 키
- 분봉 선택 (3/5/15분), 백테스트 기간 (1/3/7일), 매매시간 (기본 08~16시)
- 최소 거래대금 필터

## 기술 스택
- 단일 HTML 파일 (counter-trade5와 동일 패턴)
- Lightweight Charts (CDN) for 캔들차트
- counter-trade5의 다크 테마 스타일 재사용
- Bithumb v1 REST API
