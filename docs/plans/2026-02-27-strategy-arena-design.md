# Strategy Arena (counter-trade5.html) 설계

## 개요
6개 전략을 백테스트로 경쟁시켜 최고 성능 전략을 찾고, 해당 전략으로 실전 자동매매를 실행하는 올인원 페이지.

## 전략 목록
1. RSI 반등: RSI(14)≤30 매수, ≥70 매도
2. BB 반등: BB하단+양봉 매수, BB중앙 매도
3. EMA 크로스: EMA(5/20) 골든크로스 매수, 데드크로스 매도
4. MACD 크로스: MACD>시그널 매수, MACD<시그널 매도
5. Stochastic RSI: K<20에서 K>D 매수, K>80에서 K<D 매도
6. 복합(RSI+BB+EMA): 3조건 중 2개 이상 매수

## UI
- 탭: [전략 경쟁] [실전 매매]
- 백테스트: 전략별 순위 테이블 + 상세 히스토리
- 실전: 선택 전략으로 자동매매

## 매도 조건 (공통)
- 목표 수익률 / 시간 초과 / 전략별 기술적 매도
